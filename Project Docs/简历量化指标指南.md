#### 2. 检索延迟优化

**运行性能测试：**
```python
# 创建文件: tests/evaluation/benchmark_latency.py
import time
from app.retriever.factory import get_rag_retriever
from app.config import config

def benchmark_retrieval_latency(mode: str, num_queries: int = 100):
    """测试检索延迟"""
    config.rag_mode = mode
    retriever = get_rag_retriever()
    
    test_queries = [
        "CPU使用率过高怎么排查？",
        "内存泄漏如何定位？",
        "数据库连接池满了怎么办？",
        # ... 添加更多测试查询
    ]
    
    latencies = []
    for _ in range(num_queries):
        query = test_queries[_ % len(test_queries)]
        start = time.time()
        retriever.retrieve(query, top_k=3)
        latencies.append((time.time() - start) * 1000)  # ms
    
    return {
        "p50": sorted(latencies)[len(latencies)//2],
        "p95": sorted(latencies)[int(len(latencies)*0.95)],
        "p99": sorted(latencies)[int(len(latencies)*0.99)],
        "avg": sum(latencies) / len(latencies)
    }

# 运行对比
basic_latency = benchmark_retrieval_latency("basic")
enhanced_latency = benchmark_retrieval_latency("enhanced")
print(f"Basic P95: {basic_latency['p95']:.2f}ms")
print(f"Enhanced P95: {enhanced_latency['p95']:.2f}ms")
```

#### 4. 系统吞吐与并发

**压力测试脚本：**
```python
# 创建文件: tests/evaluation/benchmark_throughput.py
import asyncio
import time
from app.services.rag_agent_service import rag_agent_service

async def benchmark_concurrent_queries(num_concurrent: int = 50):
    """测试并发查询吞吐"""
    test_query = "CPU使用率过高怎么排查？"
    
    async def single_query(i):
        start = time.time()
        await rag_agent_service.query(test_query, session_id=f"bench_{i}")
        return time.time() - start
    
    start_time = time.time()
    tasks = [single_query(i) for i in range(num_concurrent)]
    latencies = await asyncio.gather(*tasks)
    total_time = time.time() - start_time
    
    return {
        "total_queries": num_concurrent,
        "total_time": total_time,
        "qps": num_concurrent / total_time,
        "avg_latency": sum(latencies) / len(latencies),
        "p95_latency": sorted(latencies)[int(len(latencies)*0.95)]
    }

# 运行
result = asyncio.run(benchmark_concurrent_queries(50))
print(f"QPS: {result['qps']:.2f}")
print(f"P95 Latency: {result['p95_latency']:.2f}s")
```

   - P95响应延迟 <X.Xs

## 注意事项

### ⚠️ **诚实原则**

1. **只使用真实数据**：所有量化指标必须来自实际运行的评估
2. **明确测试条件**：说明测试集规模、评估方法（如"80条标注测试集，RAGAs框架"）
3. **合理对比基线**：对比必须公平（如 basic vs. enhanced，而非 vs. 随机猜测）
4. **避免夸大**：如果提升不明显，诚实说明或换其他指标

### ✅ **最佳实践**

1. **优先使用相对提升**："提升35%"比"从0.65到0.88"更直观
2. **结合绝对值**：重要指标同时给出绝对值和相对提升
3. **说明评估方法**：提到使用的框架（RAGAs）和测试集规模
4. **业务价值优先**：技术指标要关联业务影响（如MTTR降低）
